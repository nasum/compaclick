# AGENTS.md

## Rule

1. All source code MUST be written in English.
2. All commit messages MUST be written in Japanese.
3. All commit messages SHOULD be 50 characters or less.
4. Write tests for all source code.
5. Run `go test` before committing.
6. Run `go vet` before committing.
7. Run `goimports` before committing.
8. Run `gofmt` before committing.
9. Run `staticcheck` before committing.
10. Run `golangci-lint run` before committing.
11. Save file with CRLF line endings.
